# MotionBus_AI (MotionAG) 🚍

Μια προηγμένη εφαρμογή παρακολούθησης λεωφορείων σε πραγματικό χρόνο, σχεδιασμένη για να προσφέρει την καλύτερη εμπειρία μετακίνησης. Το σύστημα συνδυάζει δεδομένα GTFS-Realtime με μια σύγχρονη και γρήγορη διεπαφή χρήστη.

## ✨ Κύριες Δυνατότητες

### 📍 Παρακολούθηση σε Πραγματικό Χρόνο (Live Map)
- **Ζωντανή Εικόνα**: Δείτε τη θέση όλων των ενεργών λεωφορείων στο χάρτη με αυτόματη ανανέωση κάθε 5 δευτερόλεπτα.
- **Λεπτομέρειες Οχημάτων**: Κάντε κλικ σε οποιοδήποτε λεωφορείο για να δείτε τη διαδρομή του, την επόμενη στάση και την καθυστέρηση (αν υπάρχει).
- **Προσαρμογή Χάρτη**: Επιλογή μεταξύ Light, Dark και Satellite modes.

### 🧠 Έξυπνος Σχεδιασμός Διαδρομής (Smart Trip Planning)
- **Πολυκριτηριακή Αναζήτηση**: Βρείτε την καλύτερη διαδρομή βάσει της ταχύτητας, της ελάχιστης πεζοπορίας ή των λιγότερων μετεπιβιβάσεων.
- **Νυχτερινά Λεωφορεία**: Επιλογή συμπερίληψης ή εξαίρεσης νυχτερινών γραμμών.
- **Υπολογισμός Απόστασης**: Αυτόματος υπολογισμός περπατήματος προς και από τις στάσεις.

### 🔔 Προηγμένο Σύστημα Ειδοποιήσεων
- **Ειδοποιήσεις Άφιξης**: Ρυθμίστε ειδοποίηση για να ενημερωθείτε όταν το λεωφορείο πλησιάζει στην επιλεγμένη σας στάση (π.χ. 5 λεπτά πριν).
- **Υποστήριξη iOS & Android**: Πλήρης συμβατότητα με Web Push Notifications, βελτιστοποιημένη ειδικά για τις αυστηρές απαιτήσεις της Apple.
- **Διαχείριση Ειδοποιήσεων**: Εύκολη προβολή και διαγραφή των ενεργών ειδοποιήσεων από το κεντρικό πάνελ.

### 🕒 Δρομολόγια και Αφίξεις
- **Εκτιμώμενος Χρόνος Άφιξης (ETA)**: Ακριβείς προβλέψεις για το πότε θα φτάσει το επόμενο λεωφορείο σε κάθε στάση.
- **Κοντινές Στάσεις**: Αυτόματος εντοπισμός των πλησιέστερων στάσεων βάσει της τρέχουσας τοποθεσίας σας.
- **Πλήρη Δρομολόγια**: Προβολή όλων των προγραμματισμένων διαδρομών για την ημέρα.

### ⭐ Αγαπημένα και Εξατομίκευση
- **Αποθηκευμένες Διαδρομές**: Αποθηκεύστε τις συχνές σας αναζητήσεις για πρόσβαση με ένα κλικ.
- **Αγαπημένες Στάσεις**: Παρακολουθήστε τις αγαπημένες σας στάσεις απευθείας από την αρχική οθόνη.

### 📱 Τεχνολογία PWA & Προσβασιμότητα
- **Εγκατάσταση (PWA)**: Εγκαταστήστε την εφαρμογή στο κινητό σας σαν κανονικό app χωρίς να περάσετε από το App Store/Play Store.
- **Προσβασιμότητα (Accessibility)**: Ειδικό widget με λειτουργίες για άτομα με μειωμένη όραση, ενισχυμένη αντίθεση και υποστήριξη screen readers.
- **Offline Support**: Βασικές λειτουργίες και προσωρινή αποθήκευση δεδομένων για χρήση ακόμη και με αδύναμη σύνδεση.

## 🛠️ Τεχνικά Χαρακτηριστικά

- **Frontend**: React + TypeScript + Vite.
- **Styling**: Tailwind CSS & shadcn/ui.
- **Backend**: Supabase Edge Functions (Edge Computing).
- **Real-time Data**: GTFS / GTFS-Realtime Integration.
- **Maps**: Mapbox / Leaflet integration.

## 🚀 Γρήγορη Εκκίνηση

1. **Εγκατάσταση Εξαρτήσεων**: `npm install`
2. **Εκκίνηση Development Server**: `npm run dev`
3. **Build για Παραγωγή**: `npm run build`
