# 🌐 Άνοιγμα του App στο Browser

## 🚀 Τι Έγινε:

Ξεκίνησα το dev server στο background.

---

## 📝 Πώς να Ανοίξεις το App:

### Επιλογή 1: Αυτόματο (Αν τρέχει το dev server)

1. Άνοιξε browser (Chrome, Edge, Firefox)
2. Πήγαινε στο: **http://localhost:8080**
3. ✅ Θα πρέπει να ανοίξει το app!

### Επιλογή 2: Αν δεν λειτουργεί το 8080

Δοκίμασε: **http://localhost:5173** (default Vite port)

### Επιλογή 3: Production URL

Αν το app είναι deployed:
- **GitHub Pages:** `https://kotsiosla.github.io/MotionBus_AI/`
- **Lovable:** (αν έχεις custom domain)

---

## 🔍 Αν Δεν Ανοίγει:

1. **Έλεγξε αν τρέχει το dev server:**
   - Κάνε click στο terminal
   - Θα πρέπει να βλέπεις: `Local: http://localhost:8080`

2. **Αν δεν τρέχει, ξεκίνησε το:**
   ```bash
   cd C:\Users\kotsi\.cursor\worktrees\chargecyprus\vzj\motionbus
   npm run dev
   ```

3. **Άνοιξε browser:**
   - Chrome/Edge: `http://localhost:8080`
   - Firefox: `http://localhost:8080`

---

## 🎯 Μετά:

Όταν ανοίξει το app:
1. Πήγαινε στο map
2. Βρες μια στάση
3. Κάνε click πάνω της
4. Κάνε click στο κουμπί 🔔 (Bell icon)
5. Επιτρέψτε notifications

---

**Άνοιξε το browser και πήγαινε στο `http://localhost:8080`! 🚀**
